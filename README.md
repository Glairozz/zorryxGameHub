# Game Hub

A collection of web-based games in one place. Browse games, read descriptions and instructions, then click to play on the game's website.

## Prerequisites

- Node.js 18+ and npm

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
src/
├── app/            # Pages (Home, Games, About, Contact)
├── components/     # UI components (Navbar, Hero, GameCard, etc.)
├── data/           # Game data definitions
└── lib/            # Utility functions
```

## Tech Stack

Next.js, TypeScript, Tailwind CSS, Framer Motion, shadcn/ui, Lucide React
